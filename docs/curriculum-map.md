## Naming rule

Every curated directory and notebook uses a descriptive lowercase name. The original course unit remains in its README and in the [notebook index](notebook-index.md).

## Completion rule

A unit is counted only when the repository contains substantive executable evidence. Zero-byte notebook scaffolds for GoingDeeper GD09 and MainQuest Quest03–Quest05 were removed, along with misplaced exact duplicates of GD07 notebooks.
